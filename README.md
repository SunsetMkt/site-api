# site-api
Open source API to handle some tasks.

## Do NOT run this on a server!
**This script is designed to be run on Vercel as a Serverless Function. There may be security issues if you run this on a server.**

## API建构原则
* 不保存用户数据
* 尽可能使用Serverless
* 可以在本地处理就不要调用API
* 来自第三方提交的代码保留原始行为
* 除token保密需要，尽可能开源透明
